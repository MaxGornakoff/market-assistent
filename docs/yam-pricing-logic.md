# Логика расчета "Рекомендуемая цена на Яндекс Маркете" и "Расходы Маркета"

## 1. Формулы и техническое описание

### Рекомендуемая цена на ЯМ (recommended_market_price)

**Формула:**

$$
\text{Рекомендуемая цена} = \frac{\text{Себестоимость} + \text{Желаемая наценка} + \text{Расходы Маркета}}{1 - \text{Суммарная комиссия ЯМ}}
$$

- **Себестоимость** — закупочная цена товара (из ERP или варианта, если у товара нет своей цены)
- **Желаемая наценка** — сумма, которую хотим заработать (может быть фиксированной или процентной)
- **Расходы Маркета** — все расходы, связанные с продажей на ЯМ (логистика, хранение, упаковка и т.д.)
- **Суммарная комиссия ЯМ** — процент комиссии Яндекса (например, 0.07 для 7%)

#### Алгоритм расчёта:
1. Получаем себестоимость товара (ERP, fallback на вариант).
2. Считаем расходы Маркета (см. ниже).
3. Складываем себестоимость, наценку, расходы.
4. Делим на (1 - комиссия ЯМ).

---

### Расходы Маркета (market_service_cost)

**Формула:**

$$
\text{Расходы Маркета} = \text{Логистика} + \text{Хранение} + \text{Упаковка} + \text{Прочие расходы}
$$

- **Логистика** — стоимость доставки (рассчитывается по тарифам ЯМ, зависит от габаритов и веса)
- **Хранение** — стоимость хранения на складе ЯМ (по тарифам, зависит от объёма и времени)
- **Упаковка** — стоимость упаковки (фиксированная или по тарифу)
- **Прочие расходы** — дополнительные расходы (например, возвраты)

#### Алгоритм расчёта:
1. Проверяем наличие всех необходимых данных (размеры, вес, тарифы).
2. Если данных не хватает — используем безопасные значения по умолчанию (например, минимальные габариты).
3. Считаем каждую составляющую по тарифам ЯМ.
4. Складываем все расходы.

## 2. Простыми словами с примером

### Пример:
- Себестоимость: 1000 ₽
- Желаемая наценка: 300 ₽
- Комиссия ЯМ: 7% (0.07)
- Логистика: 120 ₽
- Хранение: 10 ₽
- Упаковка: 5 ₽
- Прочие расходы: 0 ₽

**Расходы Маркета:**
120 + 10 + 5 + 0 = 135 ₽

**Рекомендуемая цена:**
(1000 + 300 + 135) / (1 - 0.07) = 1435 / 0.93 ≈ 1544 ₽

**Пояснение:**
- Мы хотим заработать 300 ₽ с товара, который стоит нам 1000 ₽.
- На ЯМ есть комиссия 7% и дополнительные расходы (логистика, хранение, упаковка).
- Чтобы покрыть все расходы и получить желаемую прибыль, рекомендуемая цена для ЯМ должна быть примерно 1544 ₽.

---

### Если не хватает данных
Если, например, не указаны габариты товара, для расчёта логистики и хранения используются безопасные минимальные значения (например, 10×10×1 см, 0.1 кг). В этом случае в интерфейсе будет предупреждение о том, что расчёт выполнен с допущениями.

## 3. Кратко
- Все реальные данные есть → расчёт точный.
- Чего-то не хватает → расчёт с допущениями, показываем предупреждение и список недостающих данных.
